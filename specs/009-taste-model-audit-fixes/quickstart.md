## Verification Scenarios

### Scenario 1 — Onboarding EMA (Fix 1)

```python
; Cold-start user, price_range=low place → onboarding confirmation (gain +1.2)
; price_comfort starts at 0.5, observation for "low" = 1.0
; alpha for price_comfort = 0.15
; alpha_gain = 0.15 * 1.2 = 0.18
; v_new = 0.18 * 1.0 + (1 - 0.18) * 0.5 = 0.18 + 0.41 = 0.59
; Result: price_comfort > 0.5

; Dismissal (gain -0.8) for same place
; alpha_gain = 0.15 * 0.8 = 0.12
; v_new = 0.5 - 0.12 * (1.0 - 0.5) = 0.5 - 0.06 = 0.44
; Result: price_comfort < 0.5
```

**Command**: `poetry run pytest tests/core/taste/test_service.py::test_handle_onboarding_signal_updates_taste_vector -v`
